An Arduino-powered robot that solves a rubiks cube.

How to use:
1. Flash Cube_Solver.cpp to Arduino
2. Run Send_Cube_State.py

Dependencies:
- pyserial
- arduino ide...etc...
- python 2.7 or later
- tkinter
